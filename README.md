# 概要
以下の udemy のコースを参考に作成した web アプリケーション

* [「設計を理解しながらWebアプリを開発！【はじめてのWeb開発】【Flask】」](https://www.udemy.com/course/flask_memo/learn/lecture/33731922#overview)

ユーザー認証付きのメモ共有アプリケーションであり、タイトル付きメモの保存、編集、削除ができる。

## 環境
  * フロントエンド:
    * HTML
  * バックエンド:
    * 言語: Python3(v3.12.3)
    * フレームワーク, ライブラリ:
      * Flask(v3.0.3)
      * Werkzeug(v3.0.2)
    * DB: SQLite(v3.45.3)
